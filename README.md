# financial-dashboard-analysis
Financial performance dashboard analyzing revenue, expenses, and profitability using Excel
Project Overview
This project analyzes financial performance using key metrics such as revenue, expenses, profit, profit margin, and month-over-month growth. An interactive Excel dashboard was created to visualize trends and support business decision-making.

## 📊 Key Features
- Revenue vs Expenses analysis
- Profit trend visualization
- Profit margin evaluation
- Month-over-Month (MoM) growth tracking
- KPI summary dashboard

## 📈 Key Insights
- Revenue shows overall growth with fluctuations
- Profitability is affected by inconsistent expense patterns
- Mid-year performance decline highlights cost inefficiencies
- Business shows recovery towards the end of the year

## 🛠 Tools Used
- Microsoft Excel

## 📂 Files Included
- Excel Dashboard
- Financial Analysis Report (PDF)
- Dashboard Screenshots

Simulated dataset created for financial analysis purposes.
